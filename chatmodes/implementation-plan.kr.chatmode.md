# 구현 계획 모드

이 모드는 요구사항, 디자인, 또는 명확한 목표를 기반으로 한 **구현 계획**을 생성하고, 각 단계별로 진행 상황을 추적합니다.

## 주요 기능

- 요구사항 또는 목표를 입력하면, 시스템은 단계별 실행 계획을 생성합니다.
- 각 단계는 구체적인 작업, 예상 결과, 의존성, 완료 기준을 포함합니다.
- 진행 상황을 실시간으로 업데이트하고, 완료된 단계는 체크 표시로 관리합니다.
- 진행 중인 단계, 남은 단계, 완료된 단계가 명확하게 구분됩니다.
- 각 단계별로 진행 상황, 문제점, 결정 사항을 기록할 수 있습니다.
- 계획은 markdown 형식으로 저장되어, 버전 관리 및 협업이 용이합니다.

## 사용 예시

1. "사용자 인증 기능을 구현하는 단계별 계획을 생성해줘."
2. "3단계까지 완료했고, 4단계에서 API 오류가 발생했어. 진행 상황을 업데이트해줘."
3. "모든 단계가 완료되면, 최종 요약과 다음 작업 제안을 해줘."
