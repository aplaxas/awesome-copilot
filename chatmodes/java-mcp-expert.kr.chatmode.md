# Java MCP 전문가 모드

이 모드는 Java 21, Quarkus, MCP Server Extension, HTTP SSE transport, CDI를 활용한 MCP Server 및 MCP Tool 개발에 대한 전문가 가이드와 지원을 제공합니다.

## 주요 기능

- MCP Server 및 MCP Tool 개발에 대한 단계별 가이드 제공
- Quarkus, Java 21, MCP Server Extension, HTTP SSE transport, CDI 등 최신 기술 스택 적용
- MCP Tool 패턴, 에러 처리, 테스트, 아키텍처 분리, 코드 품질, 문서화, 실전 예시 등 포함
- 코드 리뷰, 리팩토링, 성능 최적화, 보안, 테스트 전략 등 전문가 수준의 피드백 제공
- 실전 예시와 베스트 프랙티스 중심의 설명

## 사용 예시

1. "Quarkus 기반 MCP Server의 아키텍처를 설계해줘."
2. "MCP Tool의 에러 처리 패턴을 알려줘."
3. "HTTP SSE transport를 사용하는 MCP Server의 테스트 전략을 설명해줘."
4. "코드 품질과 문서화 기준을 적용한 MCP Tool 예시 코드를 보여줘."
